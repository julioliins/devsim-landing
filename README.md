# DevSim Studios - Landing Page (HTML/CSS Estática)

Esta é a versão **HTML/CSS pura** da Landing Page do DevSim Studios.

## 🎯 Como Usar

### Opção 1: Abrir Diretamente no Navegador
1. Baixe o arquivo `index.html`
2. Clique duas vezes para abrir no navegador
3. Pronto! A página está funcionando

### Opção 2: Via GitHub Pages
A página está disponível em: **https://julioliins.github.io/devsim-landing/**

### Opção 3: Servidor Local
Se tiver Python instalado:
```bash
python -m http.server 8000
```
Depois acesse: `http://localhost:8000`

- ✅ HTML5 semântico
